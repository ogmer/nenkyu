# 年間休日計算ツール

![年間休日計算ツール](https://github.com/yourusername/annual-holiday-calculator/raw/main/public/screenshot.png)

## 概要 (Overview)

年間休日計算ツールは、日本の勤務体系における年間休日数を簡単に計算できるウェブアプリケーションです。ユーザーは勤務日数や各種休暇日数を入力することで、年間の休日数を自動計算できます。企業の年間休日数の確認や、転職先の休日数の比較などにご活用いただけます。

## 機能 (Features)

**日本語**

- 週の勤務日数に基づく週末休日の自動計算
- 祝日数の設定
- 年末年始休暇の設定
- 夏季休暇の設定
- 特別休暇の設定
- 休日出勤日数の控除
- 計算結果のSNSシェア機能（Twitter、Facebook）
- ライト/ダークモード切替
- 入力値のローカルストレージ保存
- レスポンシブデザイン（モバイル対応）
