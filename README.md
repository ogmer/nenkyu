# 年間休日計算ツール (Annual Holiday Calculator)

![年間休日計算ツール](https://github.com/yourusername/annual-holiday-calculator/raw/main/public/screenshot.png)

## 概要 (Overview)

**日本語**

年間休日計算ツールは、日本の勤務体系における年間休日数を簡単に計算できるウェブアプリケーションです。ユーザーは勤務日数や各種休暇日数を入力することで、年間の休日数を自動計算できます。企業の年間休日数の確認や、転職先の休日数の比較などにご活用いただけます。

**English**

The Annual Holiday Calculator is a web application that allows users to easily calculate the total number of holidays in a Japanese work calendar. Users can input their working days and various holiday types to automatically calculate their annual holiday count. It's useful for checking company holiday policies or comparing potential employers.

## 機能 (Features)

**日本語**

- 週の勤務日数に基づく週末休日の自動計算
- 祝日数の設定
- 年末年始休暇の設定
- 夏季休暇の設定
- 特別休暇の設定
- 休日出勤日数の控除
- 計算結果のSNSシェア機能（Twitter、Facebook）
- ライト/ダークモード切替
- 入力値のローカルストレージ保存
- レスポンシブデザイン（モバイル対応）

**English**

- Automatic calculation of weekend days based on weekly work schedule
- Configuration of national holidays
- Year-end holiday settings
- Summer vacation settings
- Special holiday settings
- Deduction of overtime work days
- Social media sharing of results (Twitter, Facebook)
- Light/dark mode toggle
- Local storage of input values
- Responsive design (mobile-friendly)
