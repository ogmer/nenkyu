# 年間休日計算ツール

週の勤務日数と各種休日を入力するだけで、年間休日数を計算できます。祝日数はAPIから自動取得し、土日との重複も考慮します。

## できること

週5日勤務か週6日勤務かを選んで、年末年始やお盆などの休暇日数を入力するだけで、年間何日休みがあるか分かります。計算結果はTwitterやFacebookでシェアできます。

## 技術

Next.js 15とTypeScriptで作っています。スタイリングはTailwind CSS v4です。

## 計算の仕組み

```
年間休日数 = 週末の日数 + 祝日数 + 年末年始休暇 + 夏季休暇 + 特別休暇 - 休日出勤日数
```

週末の日数は週の勤務日数から計算します。週5日勤務なら週末は2日×52週で104日です。祝日数は平日のみカウントして、土日と重なる祝日は除いています。

## 使い方

1. 週の勤務日数を選ぶ
2. 祝日数を確認（自動入力されます）
3. 年末年始、夏季休暇、特別休暇の日数を入力
4. 休日出勤があれば入力
5. 年間休日数が表示されます

## ローカルで動かす

```bash
npm run dev
```

http://localhost:3000 で確認できます。

## デプロイ

Vercelにそのままデプロイできます。

## 注意

これは簡易計算ツールです。振替休日には対応していません。正確に知りたい場合は実際のカレンダーを確認してください。入力データはブラウザに保存されます。
